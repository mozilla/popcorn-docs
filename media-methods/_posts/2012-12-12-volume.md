# volume( \[value\] ) #

## Purpose ##

Get or Set the volume of a Popcorn instance.

Calling the `volume( value )` method will emit a volumechange event.

## Options ##

* **value** \[Number\] - Passing a `value` (0 - 1.0) parameter will update the media's current volume.
