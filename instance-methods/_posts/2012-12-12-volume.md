# volume( [value] ) #

## Purpose ##

Get or Set the volume of a Popcorn instance.

 - Calling the `.volume( value )` method will trigger a `**volumechange**` event.

## Options ##

* `**value**` - Passing a `value` (0 - 1.0) parameter will update the media's current volume.
